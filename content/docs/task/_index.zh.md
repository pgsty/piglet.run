**任务**文档提供面向目标的具体操作指南。

## 如何使用

每个任务指南专注于完成一个具体目标：

- **问题导向**：从你想要实现的目标出发
- **逐步操作**：清晰的步骤说明
- **实战场景**：真实世界的应用场景

## 主题

| 主题 | 描述 |
|------|------|
| [备份](/docs/task/backup/) | 备份数据库和文件 |

| [克隆](/docs/task/clone/) | 克隆数据库或环境 |
| [部署](/docs/task/deploy/) | 部署 Web 应用 |
| [SSL](/docs/task/ssl/) | 配置 SSL 证书 |

| [监控](/docs/task/monitor/) | 设置告警和监控 |
| [升级](/docs/task/upgrade/) | 升级 Piglet Run |
