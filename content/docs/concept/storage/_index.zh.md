## JuiceFS 共享存储

Piglet Run 使用 **JuiceFS** 提供基于 PostgreSQL 的分布式文件系统。

## 为什么选择 JuiceFS？

- **POSIX 兼容**：像普通文件系统一样工作
- **数据库后端**：数据存储在 PostgreSQL
- **快照支持**：时间点恢复
- **多用户**：跨会话共享工作空间

## 工作原理

```
┌────────────────────────────────────────────┐
│              应用层                         │
│   (VS Code, Jupyter, Claude Code 等)       │
└──────────────────┬─────────────────────────┘
                   │ POSIX API
                   ▼
┌────────────────────────────────────────────┐
│               JuiceFS FUSE                 │
│         (用户空间文件系统)                  │
└──────────────────┬─────────────────────────┘
                   │
        ┌──────────┴──────────┐
        │                     │
        ▼                     ▼
┌───────────────┐    ┌───────────────┐
│    元数据     │    │   数据块      │
│  (PostgreSQL) │    │  (PostgreSQL) │
└───────────────┘    └───────────────┘
```

## 特性

| 特性 | 描述 |
|------|------|
| **透明** | 像本地文件系统一样使用 |
| **持久** | 数据存储在数据库中 |
| **并发** | 多用户/Agent 同时访问 |
| **快照** | 时间点恢复 |
