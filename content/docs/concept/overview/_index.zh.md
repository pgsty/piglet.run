## 什么是 Piglet Run？

**Piglet Run**（小猪运行时）是 Pigsty 的轻量运行时环境，专为 **AI Web Coding** 打造的**云端编码沙箱**。它将 PostgreSQL 数据库、JuiceFS 分布式存储、VS Code、JupyterLab 等工具整合为一体。

## 为什么选择 Piglet Run？

在 AI 辅助开发的时代，开发者需要：

- **即用型开发环境**，开箱即用
- **强大的数据库**，所有扩展都可用
- **安全的实验环境**，轻松回滚
- **无缝协作**，人与 AI Agent 协同工作

Piglet Run 将这一切整合在一个软件包中。

## 核心特性

| 特性 | 说明 |
|------|------|
| 🤖 **AI 编码** | 预装 Claude Code、OpenCode、VS Code、Jupyter |
| 🐘 **数据全能** | PostgreSQL 18 + 400+ 扩展 |
| 💾 **共享存储** | JuiceFS 将工作目录存入数据库 |
| ⏱️ **时光机器** | 数据库 PITR + 文件系统快照 |
| 🔀 **瞬间克隆** | Copy-on-Write 零拷贝克隆 |
| 🌐 **一键上线** | 内置 Nginx，自动 SSL |
| 📊 **全栈监控** | VictoriaMetrics + Grafana |
| 🇨🇳 **国内畅通** | 全球 CDN + 国内镜像 |

## 适用人群

- **独立开发者**：需要强大开发环境
- **开发团队**：需要共享开发基础设施
- **AI 开发者**：使用 Claude Code 等工具
- **数据科学家**：使用 PostgreSQL 和 Jupyter
- **学习者**：探索 PostgreSQL 和 Web 开发
